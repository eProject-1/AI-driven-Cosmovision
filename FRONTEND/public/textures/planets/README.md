Planet texture drop-in folder.

The 3D planet viewer checks this folder before trying remote texture URLs. Add equirectangular texture maps with these names:

- `earth_color.jpg`, `earth_normal.jpg`, `earth_clouds.png`
- `mars_color.jpg`, `mars_normal.jpg`
- `saturn_color.jpg`, `saturn_rings.png`
- `mercury_color.jpg`, `venus_color.jpg`, `jupiter_color.jpg`, `uranus_color.jpg`, `neptune_color.jpg`

`.jpg`, `.webp`, and `.png` are supported. Use 2:1 equirectangular maps for planet surfaces.
